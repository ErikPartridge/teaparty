Use the following commands to manage cards, case insensitive

  • DEAL 1XXXXXX - deal a card, or two if this CID hasn't been used before
  • DROP ZZ 1XXXXXX - drop the first instance of the specified card (ZZ) from the user with that CID
  • LIST - show all users and their cards
  • SHOW 1XXXXXX - generate a message you can send to that user with their hand
